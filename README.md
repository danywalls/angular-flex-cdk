# Angular CDK Layout and Flex for Responsive Websites

## Project Summary
This project showcases how to replace float-based layouts with Flexbox and use the Angular CDK Layout for responsive design without excessive media queries.

## How to Run
### Installation
1. `npm install` 

### Running
1. `ng serve` 
2. View the responsive layout in the browser.
